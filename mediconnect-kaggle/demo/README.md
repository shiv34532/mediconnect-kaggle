# Demo Guidelines

## Recording Your 5-Minute Video

### Setup
- Quiet room with good lighting
- Screen recording software (OBS, Loom, QuickTime)
- Microphone test before recording

### Required Elements
1. Problem statement with real-world impact
2. Architecture explanation with visual diagram
3. Live code demo (not slides)
4. Security features demonstration
5. Deployment walkthrough

### Tips
- Practice timing—5 minutes goes fast
- Have code pre-loaded to avoid waiting
- Use zoom for important code sections
- Show errors and how system handles them

### Upload
- YouTube: Upload as unlisted or public
- Copy link for Kaggle Media Gallery
- Verify link works in incognito window
